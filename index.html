Simple Genetic Algorithm

Paramètres

Définissez les paramètres de l'algorithme génétique et cliquez sur "Démarrer" pour l'exécuter.

La taille de la population est le nombre d'individus dans la population.

Le nombre maximum de générations est le nombre maximum de générations pour exécuter l'algorithme.

Le taux de croisement est la probabilité que deux individus croisent leurs gènes.

Le taux de mutation est la probabilité qu'un gène mute.

Le nombre d'appels est le nombre d'appels à la fonction objectif (fonction d'évaluation).
